# The Trump Files - Project Overview

## Purpose

A comprehensive, data-driven web application serving as an "encyclopedia" cataloging controversial moments and actions of the 45th U.S. President. The site provides:

- **Data Archive**: ~775 documented entries with sources, fact-checks, and AI analysis
- **Interactive Visualizations**: Multi-chart data visualizer (pie, bar, line, radar, etc.)
- **Advanced Filtering**: Search, category filters, phase filters, pagination
- **Analytical Tools**: AI-powered scoring across 8 dimensions (danger, authoritarianism, lawlessness, insanity, absurdity, etc.)
- **Public Archive**: Accessible, searchable database for historical analysis

## Core Philosophy

- **Data-Driven**: Every entry is sourced, verified, and scored
- **Transparent**: Sources cited, methodology documented
- **Accessible**: Free public access to comprehensive archive
- **Satirical**: Humor used to make serious data more engaging
- **Evidence-Based**: Rigorous fact-checking and verification

## Target Audience

- General public seeking information
- History students
- Anyone researching the Trump presidency
